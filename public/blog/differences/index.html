Typography

Tip

  • If you are looking for more options, Poppins is an alternative to Lexend we considered for Xeo. Design

  • Spacing: Xeo adjusts spacing around various elements, particularly page headers and the search box, which is relocated to the bottom of pages to reduce layout shift when search results appear.

  • Color: Xeo has a foundation that will lead to eventual support of Rainbow Mode powered by Chromagen, like Xeo‚Äôs predecessor eleventeen. Chromagen generated Xeo‚Äôs current color schemes, but has not yet been wired up to Xeo to the degree it has been integrated into eleventeen.
